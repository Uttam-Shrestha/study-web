🏗️ Computer Architecture (Von Neumann)

Computer architecture means how the parts of a computer are arranged and how they talk to each other. The most famous design is the Von Neumann Architecture.

Main Parts of Von Neumann Architecture

  • Input Unit: Brings data inside (keyboard, mouse).
  • Central Processing Unit (CPU): The brain. Has two parts:
    • Control Unit (CU): Directs all operations like a traffic police.
    • Arithmetic Logic Unit (ALU): Does math (+ , –) and logic (AND, OR).
  • Memory Unit: Stores data and instructions (RAM).
  • Output Unit: Shows results (monitor, printer).
🔑 Key idea: Both data and instructions are stored in the same memory. This is called stored program concept.

Example: You type 2+2 on keyboard (input) → CPU calculates (ALU) using instructions from memory → Monitor shows 4 (output).

🧠 Buffer, Cache, RAM, and ROM

These are all types of memory used in a computer. They differ in speed, size, and purpose.

1. Buffer

A temporary waiting area in memory. It holds data while it moves from one place to another.

📌 Example: When you watch a YouTube video, the grey bar that loads ahead is stored in a buffer. It prevents the video from stopping.

2. Cache

A super-fast, small memory inside or very near the CPU. It stores copies of frequently used data so the CPU doesn't have to go to the slower RAM every time.

📌 Example: Think of it as your desk drawer. You keep your most-used pen there instead of going to the cupboard (RAM) every time.

3. RAM (Random Access Memory)

Main working memory. It is volatile – data disappears when power is off. It holds the operating system, open apps, and current work.

📌 Example: When you open Notepad and type, your text is in RAM. If the computer turns off before saving, the text is lost.

4. ROM (Read Only Memory)

Permanent memory. It is non-volatile – data stays even without power. It contains the instructions to start the computer (BIOS/firmware).

📌 Example: When you press the power button, ROM tells the computer how to wake up and load the operating system.

🎨 Types of CSS (Cascading Style Sheets)

CSS is used to style HTML pages. There are three ways to add CSS to a webpage.

1. Inline CSS

Written directly inside an HTML tag using the style attribute.

<p style="color: red; font-size: 20px;">Hello</p>

Use when: You need a quick, one-time style for a single element.

2. Internal (Embedded) CSS

Written inside the <style> tag in the <head> section of the HTML page.

<head>
                <style>
                p { color: blue; }
            </style>
        </head>

Use when: Styling a single webpage with its own unique design.

3. External CSS

Written in a separate file (like style.css) and linked to the HTML page.

<head>
            <link rel="stylesheet" href="style.css">
        </head>

Use when: You want one style sheet to control many pages (best practice).

🔢 Number System Conversion

Computers understand only binary (0 and 1). We often convert between Decimal, Binary, Octal, and Hexadecimal.

Common Number Systems

SystemBaseDigits UsedExample
Decimal100 to 925
Binary20, 111001
Octal80 to 731
Hexadecimal160 to 9, A to F19

Decimal to Binary (Example: 13)

  1. Divide 13 by 2 → quotient 6, remainder 1
  2. Divide 6 by 2 → quotient 3, remainder 0
  3. Divide 3 by 2 → quotient 1, remainder 1
  4. Divide 1 by 2 → quotient 0, remainder 1
  5. Read remainders from bottom to top: 1101

So, 13 in decimal = 1101 in binary.

Binary to Decimal (Example: 1101)

Multiply each bit by powers of 2 from right (start at 2⁰):

1×2³ + 1×2² + 0×2¹ + 1×2⁰ = 8 + 4 + 0 + 1 = 13

➖ 1's Complement and 2's Complement

These are methods to represent negative numbers in binary. Computers use 2's complement most of the time.

1's Complement

Flip all bits: change 0 to 1 and 1 to 0.

Example: Binary for 5 is 0101 (using 4 bits).
1's complement of 5 = 1010 (this represents –5 in 1's complement form).

2's Complement

Step 1: Find 1's complement (flip bits).
Step 2: Add 1 to the flipped bits.

Example: For 5 (0101):
1's complement = 1010
Add 1 = 1010 + 1 = 1011
So, –5 in 2's complement is 1011.
💡 Why 2's complement is better: There is only one representation for zero (no negative zero problem). Addition and subtraction become very simple for the CPU.

💻 Basic Programs – Sum & Formula Type

These are simple programs that take input, do a calculation using a formula, and show output. They are usually the first programs you learn.

1. Sum of Two Numbers

Formula: sum = a + b

// C program example
            #include <stdio.h>
            int main() {
                int a = 10, b = 20;
                int sum = a + b;
                printf("Sum is: %d", sum);  // Output: Sum is: 30
                return 0;
            }

2. Area of a Circle

Formula: area = π × r²

float radius = 5.0;
                float area = 3.1416 * radius * radius;
                printf("Area: %.2f", area);  // Output: Area: 78.54

3. Simple Interest

Formula: SI = (P × T × R) / 100

float P = 1000, T = 2, R = 5;
                    float SI = (P * T * R) / 100;
                    printf("Simple Interest: %.2f", SI);  // Output: 100.00

4. Celsius to Fahrenheit

Formula: F = (C × 9/5) + 32

float celsius = 37;
                        float fahr = (celsius * 9.0/5.0) + 32;
                        printf("Fahrenheit: %.2f", fahr);  // Output: 98.60

📊 Array – Theory and Examples

An array is a collection of items of the same data type stored in contiguous (side-by-side) memory locations. Think of it as a row of lockers, each with a number (index).

Key Features of an Array

  • Fixed size: Once you create an array of size 5, you cannot change it to size 10 easily.
  • Same data type: All elements must be of the same type (all integers, or all floats).
  • Index starts at 0: The first element is at index 0, second at index 1, and so on.
  • Random access: You can directly access any element using its index, like arr[3].

Example: Storing 5 Marks

#include <stdio.h>
                            int main() {
                                // Declare and initialize an array of 5 integers
                                int marks[5] = {85, 72, 90, 68, 88};

                                // Access and print each element
                                for(int i = 0; i < 5; i++) {
                                    printf("Student %d got %d marks.\n", i+1, marks[i]);
                                }

                                // Calculate average
                                int sum = 0;
                                for(int i = 0; i < 5; i++) {
                                    sum = sum + marks[i];
                                }
                                float avg = sum / 5.0;
                                printf("Average marks: %.2f", avg);
                                return 0;
                            }

Output: Student 1 got 85 marks... Average marks: 80.60

🎬 Components of Multimedia

Multimedia means using more than one type of media to share information. The main components are:

1. Text

The most basic part. Words, headings, paragraphs. It gives the main message.

Example: Subtitles in a video, captions on an image

2. Graphics / Images

Still pictures, drawings, photographs. They make content visually appealing.

Example: A photo of a lion in a wildlife presentation

3. Audio

Sound – speech, music, or sound effects. It adds emotion and clarity.

Example: Background music in a game, narration in a tutorial

4. Video

Moving images. A sequence of pictures shown quickly to create motion.

Example: A recorded lecture, a movie clip

5. Animation

Creating the illusion of movement by displaying drawings or models in sequence.

Example: Cartoon characters moving, a spinning 3D logo
🎯 Multimedia = Text + Graphics + Audio + Video + Animation working together.

⚙️ Types of Operating Systems and Their Functions

An Operating System (OS) is the main software that manages all hardware and software on a computer. It acts as a bridge between the user and the computer hardware.

Core Functions of an OS

  1. Process Management: Handles running programs (processes), decides which one uses the CPU and for how long.
  2. Memory Management: Allocates RAM space to different programs and frees it when done.
  3. File Management: Organizes files and folders on storage drives (creating, deleting, reading files).
  4. Device Management: Controls input/output devices (printer, keyboard, mouse) through drivers.
  5. Security: Protects data with passwords, user accounts, and access permissions.
  6. User Interface: Provides a way for users to interact – either Graphical (GUI) or Command Line (CLI).
